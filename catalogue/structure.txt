================================================================
ОПИСАНИЕ КАТЕГОРИЙ (Бильярд)
================================================================

КИИ  (~66 товаров)
  Все бильярдные кии: для пула, русского бильярда, снукера.
  Цельные и разборные. Укороченные и карбоновые.
  Комплекты из нескольких одинаковых киев (опт).
  НЕ включает: подарочные наборы (кий+тубус+мел) → Наборы.
  Теги: вид_игры, составность, длина_см, бренд

ШАРЫ  (~28 товаров)
  Все бильярдные шары: пул, русский бильярд, снукер, карамболь.
  Одиночные и в комплектах. С чистящими средствами в кейсе.
  Теги: вид_игры, размер_мм, бренд

СТОЛЫ
  Бильярдные столы (пока без товаров).
  Теги: вид_игры, размер_фт, бренд

СУКНО  (~60 товаров)
  Сукно для бильярдных столов всех размеров и составов.
  Теги: размер_стола_фт, бренд, состав

ПОКРЫВАЛА И ЧЕХЛЫ ДЛЯ СТОЛОВ  (~69 товаров)
  Покрывала (декоративные, с бахромой) и защитные чехлы
  на резинке для бильярдных столов.
  Теги: размер_стола_фт, тип (покрывало / чехол), модель

ТУБУСЫ И ЧЕХЛЫ ДЛЯ КИЕВ  (~25 товаров)
  Тубусы, футляры, кейсы и мягкие чехлы для транспортировки
  и хранения бильярдных киев.
  Теги: вид_игры, тип (тубус / чехол / футляр), кол-во_отделений

КИЕВНИЦЫ  (~25 товаров)
  Напольные и настенные стойки/полки для хранения киев.
  Теги: тип (напольная / настенная), вместимость, цвет

КОМПЛЕКТУЮЩИЕ СТОЛОВ  (~41 товар)
  Лузы, резина для бортов, сетки для луз, пелерины,
  скобы, подпятники/опоры, крюки — всё для сборки,
  ремонта и обслуживания бильярдного стола.
  Теги: тип, размер_стола_фт, материал

АКСЕССУАРЫ  (~89 товаров)
  Всё для игры и ухода, не попавшее в отдельную категорию:
  - Мел, наклейки, колпачки, инструменты для наклеек
  - Перчатки
  - Мосты (тёщи) и древки для мостов
  - Щётки, станки для чистки сукна
  - Средства ухода за шарами, муфты для чистки кия
  - Держатели мела
  - Подвесы/держатели для киев, полки для шаров
  - Треугольники
  - Тренажёры
  - Наборы мелких аксессуаров (мел + инструмент и т.п.)
  Теги: тип, вид_игры, бренд

НАБОРЫ  (~7 товаров)
  Подарочные комплекты для игрока: кий + тубус/чехол +
  мелки (+ перчатка и т.п.). Главный компонент — кий.
  Теги: вид_игры, состав_набора

ОСВЕЩЕНИЕ
  Светильники для бильярдных столов (пока без товаров).
  Теги: длина, тип
